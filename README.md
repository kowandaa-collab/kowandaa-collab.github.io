## 🗂️ File & Folder Organization

| Path | Purpose |
|------|---------|
| `index.html` | Main landing page — the first page visitors see |
| `assets/` | Static resources (images, icons, fonts) |
| `assets/css/` | Stylesheets |
| `assets/js/` | JavaScript files |
| `pages/` | Additional HTML pages (About, Projects, Contact, etc.) |
| `README.md` | This file — repository documentation |

> Files are organized so the root stays clean: only `index.html` and top-level config files live there, while everything else is sorted into named sub-folders by type.
